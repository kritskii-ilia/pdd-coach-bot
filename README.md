# PDD Coach Bot

Telegram-бот для подготовки к теоретическому экзамену на вождение (ПДД РФ). Короткие уроки, тематические тренировки, адаптивное повторение слабых мест.

**[Попробовать бота](https://t.me/RUS_PDDbot)**

## Features

- Короткие уроки по базовым темам ПДД (1-3 минуты)
- Тематическая тренировка сразу после урока
- Смешанные мини-тесты из разных тем
- Адаптивная логика: слабые темы повторяются чаще
- Прогресс: темы, вопросы, билеты, дневная цель
- Уведомления в течение дня (микро-касания)
- Импорт полной базы билетов с изображениями
- Фоновый запуск через systemd + автозапуск в Windows

## Tech Stack

- Python 3.12
- python-telegram-bot
- SQLite
- systemd
- JSON-контент (уроки, вопросы, план обучения)

## How It Works

1. Базовая тема: короткий урок
2. Закрепление: 5-7 вопросов по теме
3. После нескольких тем: смешанный тест
4. После базы: переход к билетам
5. Ошибки получают более ранний повтор

## Quick Start

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -e .[dev]
cp .env.example .env
# Заполни PDD_BOT_TOKEN в .env
python -m pdd_coach_bot.main
```
